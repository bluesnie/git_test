## 前言

- `gitbook install` 安装插件
- `gitbook serve` 启服务，并且会生成目录
- `npm run build` 生成电子书（修改了文档，记得及时执行此目录更新）
- docs 是网页电子书目录（github上指定的）
    - `gitbook serve` 生成目录前，需要把 docs 删除，不然目录里面会有这个，然后再执行 npm run build
